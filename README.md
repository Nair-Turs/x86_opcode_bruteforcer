# x86-opcode-bruteforcer

Bruteforces 1- and 2-byte opcode space for x86 architecture using Capstone disassembler.

## Quick Start

1. Create venv:
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -U pip
   pip install .

2. Run:
   opcode-scan
   
   Outputs: SAME_opcodes.json, ALL_opcodes.json
